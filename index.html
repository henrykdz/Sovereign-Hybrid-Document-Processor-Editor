✨ Key Features

✅ ISO‑compliant Export A4, Letter, Web – High-fidelity HTML & PDF output.
✅ Smart PDFs Searchable, selectable, and structurally clean –
fully editable in Adobe Acrobat, InDesign & Co.
✅ Source Swap & Highlight Flicker-free real-time rendering via WebKit injection with Syntax Highlighting for code clarity.
✅ Source Mapping Click in the preview, jump to the line: Fast navigation directly to the correct position in your document source.
✅ Smart Formatting One-click Auto‑Fix for Markdown, HTML, and CSS
to ensure perfect structural integrity (Ctrl+Shift+F).
✅ CSS Forge & Design Inspector High-performance styling engine with a dedicated Design Inspector. Instantly identifies missing classes on click and reveals Architecture Insights to master the system's styling hooks.
✅ Collision‑Free Styling User-defined CSS is strictly isolated via Layered Namespaces. Style your document with absolute freedom without ever affecting the editor's core UI.
✅ Diagnostic Linter & Radar Real-time structural validation with High‑Fidelity Inline Popups. The global Status-Bar Summary tracks the most frequent issues and provides intelligent corrective tips for an error-free workflow.
✅ Precision Slicer Engine Physical 3.78 px/mm geometry for true‑to‑scale 1:1 rendering. When in pagination mode, it accurately shifts oversized elements to subsequent pages while preserving their structural integrity. Masterful synergy between automated logic and manual page-break overrides via dedicated controls, ensuring absolute layout sovereignty and unfragmented information blocks. Includes automatic image scaling to ensure visuals always fit the page dimensions (max 100%) without breaking the layout.
✅ Custom Placeholders Minimalist YAML-based data injection for dynamic
templates and automated document generation.

🏗️ Technical Foundation

Engineered with Java 23 & Apache Maven, FlowShift combines the stability of a native desktop powerhouse with the infinite styling possibilities of modern web technologies.

Component Technology
Build System Apache Maven (Lifecycle & Dependency Management)
Engine & Runtime JavaSE-23 (Zulu) with integrated JavaFX (WebKit Core, FXML)
Markdown Core Flexmark-Java v0.64.8 (Ext: YAML, Tables, Attributes)
Rich Text RichTextFX v0.11.7
CSS Architecture Layered DOM Hierarchy (Isolated .workspace / .page / .content namespaces)
Coordinate System Physical-to-Digital Mapping (3.78 px/mm @ 96 DPI for true-to-scale 1:1 rendering)
