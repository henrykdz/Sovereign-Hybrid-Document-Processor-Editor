6.2 Hybrid Rendering & Source Mapping

The engine utilizes the Sovereign Swap mechanism. Instead of standard reloading, content is injected directly into the WebKit DOM via JavaScript/innerHTML, ensuring absolute smoothness and zero flicker.

The SovereignSourceMapper injects a unique data-fsid into every HTML element. This creates a character-accurate link between the Markdown source and the rendered preview, enabling precise bidirectional navigation.
