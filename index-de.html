🚀 Einführung: The Document Forge

Die FlowShift Sovereign Document Engine ist ein leistungsstarker Hybrid-Dokumenten-Builder. Sie überwindet die Grenzen traditioneller Editoren, indem sie Markdown, HTML5/CSS3 und YAML in einem einzigen Workflow vereint.

Es ist ein Präzisionsinstrument, das die Lücke zwischen strukturiertem Inhalt und professionellem Layout schließt – Meisterwerke mit absoluter Datensouveränität.

🧠 Kernphilosophie

✨ Hauptfunktionen

✅ Source Swap Flimmerfreie Echtzeit-Darstellung via WebKit-Injection
✅ Source Bridge Zeichengenaue bidirektionale Navigation
✅ CSS Forge Integrierte CSS-Engine für Live-Dokumentenstyling
✅ Formatter Automatische Formatierung von Markdown, HTML und CSS
✅ Linter Echtzeit-Fehlererkennung mit visuellem Feedback
✅ Syntax Highlighter Farbig hervorgehobener Code für bessere Lesbarkeit
✅ Custom Placeholders Benutzerdefinierte Platzhalter für Vorlagen und Serienbriefe
✅ ISO-konformer Export A4, Letter, Web – alles möglich

🏗️ Technische Grundlage

Entwickelt mit Java 23 & JavaFX, vereint FlowShift die Stabilität einer nativen Desktop-Powerhouse mit den unendlichen Styling-Möglichkeiten moderner Webtechnologien.

Komponente Technologie
Laufzeitumgebung JavaSE-23 (Zulu 23) mit integriertem JavaFX
UI-Framework JavaFX 23 (WebKit Core, FXML)
Markdown-Kern Flexmark-Java v0.64.8 (Ext: YAML, Tables, Attributes)
Rich Text RichTextFX v0.11.7

📖 Philosophie der Resilienz
